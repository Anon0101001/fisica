--- layout: home # articles: # excerpt_type: html --- Para el presente trabajo utilizaremos Git + Github para el trabajo colaborativo. El lenguaje de programación escogido fue Python el cual es el lenguaje en común, ahora para evitar problemas todos deben instalar y utilizar Python en su versión 3.7 ya que será la versión oficial de este proyecto. Si deciden utilizar otra versión, y sus códigos no son compatibles con el código fuente serán rechazados sus aportes al proyecto. Hemos elegido Tkinter como la librería gráfica dado a su extensa documentación y amplio número de tutoriales en internet, además de tener la ventaja de ser una librería incorporada dentro de Python. Para este proyecto en especial, recomendamos el uso de un IDE especializado, especificamente Pycharm dado a que les ayudará a corregir errores menores, mantener un formato adecuado y además de tener la ventaja de tener un sistema de control de versiones integrado.
